# Bosch Indego Calendar Card
![Home Assistant](https://img.shields.io/badge/Home%20Assistant-Custom%20Card-blue)
![Bosch Indego](https://img.shields.io/badge/Bosch-Indego-green)

A custom Home Assistant Lovelace card that displays Bosch Indego mowing schedules as a weekly calendar.

The card visualizes the configured mowing windows for each day of the week and provides a clear timeline view from 00:00 to 24:00.


## Features

- Weekly calendar view
- Visual representation of mowing windows
- Supports up to two mowing slots per day
- Current time indicator on the current day
- Optional highlighting of today's row
- Configurable colors
- Built-in Lovelace visual editor
- English and German localization
- Responsive design matching Home Assistant themes

---

## Requirements

This card requires an entity that provides the following attributes:

```yaml
monday_slot_1
monday_slot_2
tuesday_slot_1
tuesday_slot_2
wednesday_slot_1
wednesday_slot_2
thursday_slot_1
thursday_slot_2
friday_slot_1
friday_slot_2
saturday_slot_1
saturday_slot_2
sunday_slot_1
sunday_slot_2
```

Expected format:

```yaml
monday_slot_1: "08:00-12:00"
monday_slot_2: "14:00-16:00"
```

Disabled slots can contain:

```yaml
not_enabled
```

This card was designed for use with Bosch Indego calendar entities generated by the Bosch Indego Home Assistant integration.

## Installation

### HACS (Custom Repository)

1. Open HACS
2. Select **Custom Repositories**
3. Add this repository URL
4. Select category **Dashboard**
5. Install the card
6. Restart Home Assistant
7. Refresh your browser

### Manual Installation

1. Download `indego-calendar-card.js`
2. Copy it to:

```text
/config/www/indego-calendar-card.js
```

3. Add the resource:

```yaml
url: /local/indego-calendar-card.js
type: module
```

4. Reload Home Assistant.

## Configuration

### Basic Example

```yaml
type: custom:indego-calendar-card
entity: sensor.indego_calendar_slots
```

### Full Example

```yaml
type: custom:indego-calendar-card
entity: sensor.indego_calendar_slots
title: Mowing Schedule

highlight_today: true

day_color:
  - 0
  - 122
  - 61

slot_color:
  - 0
  - 122
  - 61

now_color:
  - 166
  - 206
  - 57

today_border_color:
  - 255
  - 215
  - 0
```

---

## Options

| Name | Type | Default | Description |
|--------|--------|--------|--------|
| entity | string | required | Calendar entity |
| title | string | Calendar | Card title |
| highlight_today | boolean | false | Highlight today's row |
| day_color | RGB array | [0,122,61] | Day label color |
| slot_color | RGB array | [0,122,61] | Mowing slot color |
| now_color | RGB array | [166,206,57] | Current time indicator color |
| today_border_color | RGB array | [255,215,0] | Border color for highlighted day |

## Visual Editor

The card includes a built-in Lovelace GUI editor.

You can configure:

- Entity
- Title
- Day color
- Slot color
- Current time color
- Highlight today
- Today's border color

directly from the Home Assistant dashboard editor.
